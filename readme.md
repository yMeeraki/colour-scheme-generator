# Color Scheme Generator

A simple web app that generates beautiful color palettes using The Color API.

## Features

- Pick any color using the color picker.
- Choose a color scheme mode.
- Generate 5 matching colors.
- View the HEX code for each color.

## Built With

- HTML
- CSS
- JavaScript
- The Color API

## API

https://www.thecolorapi.com/

## Preview

Select a color, choose a scheme, and click **Get Color Scheme** to generate a palette.
